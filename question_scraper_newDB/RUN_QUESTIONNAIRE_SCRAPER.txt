================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: Application won't start
SOLUTION:
  - Make sure Tesseract-OCR is installed
  - Ensure it's at: C:\Program Files\Tesseract-OCR
  - Restart computer and try again

PROBLEM: "Tesseract not found" error
SOLUTION:
  - Download and install from: https://github.com/UB-Mannheim/tesseract/wiki
  - Use default installation path
  - Restart your computer

PROBLEM: "API Disconnected" message
SOLUTION:
  - Check internet connection
  - Render.com API might be temporarily down
  - Try again in a few moments

PROBLEM: "Could not extract text"
SOLUTION:
  - Make sure question text is clear
  - Try selecting a larger area
  - Check that Tesseract is working
  - Reinstall Tesseract if needed

PROBLEM: Windows says "Unknown developer" warning
SOLUTION:
  - This is normal for self-created .exe files
  - Click "More info" then "Run anyway"
  - It's safe - you created it!
