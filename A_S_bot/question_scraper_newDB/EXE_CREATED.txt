================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: Application won't start
  REASON: Tesseract-OCR not installed
  SOLUTION:
    1. Download from: https://github.com/UB-Mannheim/tesseract/wiki
    2. Install to: C:\Program Files\Tesseract-OCR
    3. Restart computer
    4. Try again

PROBLEM: "Tesseract not found" error
  REASON: Tesseract installed to wrong location
  SOLUTION:
    1. Uninstall current Tesseract
    2. Reinstall to: C:\Program Files\Tesseract-OCR
    3. Accept default path
    4. Restart computer

PROBLEM: Takes 5-10 seconds to start
  REASON: Normal for first run (loading libraries)
  SOLUTION:
    - Wait 10 seconds
    - Subsequent runs are faster
    - This is expected behavior

PROBLEM: Application starts but window doesn't appear
  REASON: Still loading in background
  SOLUTION:
    - Wait another 5 seconds
    - Check taskbar for window
    - Try clicking on taskbar icon

PROBLEM: "API Disconnected" message
  REASON: Internet connection issue
  SOLUTION:
    - Check internet connection
    - Render.com API might be down
    - Try again in a few moments

PROBLEM: "Could not extract question text"
  REASON: OCR failure (Tesseract issue)
  SOLUTION:
    - Make sure question text is clear
    - Select larger area
    - Try clearer fonts
    - Reinstall Tesseract if needed

PROBLEM: Windows SmartScreen warning
  REASON: Unknown executable (normal for custom apps)
  SOLUTION:
    - Click "More info"
    - Click "Run anyway"
    - It's safe - you created it!
    - Won't appear next time
